# Netflix Landing Page Clone 

A responsive Netflix-style landing page built using **React** and **CSS**, inspired by the official Netflix homepage. This project focuses on clean UI design, responsiveness, and component-based development.

##  Features

- Netflix-style dark UI theme
- Fully responsive (mobile, tablet & desktop)
- Reusable React components
- Hero section with email call-to-action
- Multiple feature sections with images
- Clean and modern layout

##  Technologies Used

- React (Create React App)
- JavaScript (ES6)
- CSS (Flexbox & Media Queries)

##  Responsive Design

The layout adapts smoothly across:
- Mobile phones
- Tablets
- Desktop screens

##  How to Run the Project Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/netflix-landing-page.git

2. Navigate into the project folder:

cd netflix-landing-page


3. Install dependencies:

npm install


4. Start the development server:

npm start


The app will open in your browser at:

http://localhost:3000
